# SalaryLens

## アーキテクチャ
- UI ライブラリ: React 19
- フレームワーク: React Router v7
- スタイリング: Tailwind CSS v4
- バリデーション: zod
- ORM: Prisma
- アイコン: lucide
- ディレクトリ: コロケーション
