# ページ構成

## 概要

資産管理アプリ「SalaryLens」のページ構成を定義する。

## ページ一覧

| #   | ページ名             | routes パス       | 概要                                |
| --- | -------------------- | ----------------- | ----------------------------------- |
| 1   | ダッシュボード       | dashboard/        | 総資産、今月の収支サマリー          |
| 2   | 給料明細アップロード | payslips/upload/  | 画像/PDFアップロード、OCR or 手入力 |
| 3   | 給料明細一覧         | payslips/ (index) | 過去の明細リスト、検索/フィルター   |
| 4   | 給料明細詳細         | payslips/$id/     | 個別明細の内訳表示                  |
| 5   | 収支分析             | analytics/        | 月別/年別推移グラフ、カテゴリ別分析 |
| 6   | レポートエクスポート | reports/          | CSV/PDF出力、期間指定               |

## 採用しない機能

- **金融口座連携**: 個人アプリではAPIの使用ハードルが高いため見送り
